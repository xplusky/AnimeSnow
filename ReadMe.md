# AnimeSnow

WPF桌面下雪工具

## 安装及使用

http://leaful.com/works/leaful-anime-snow-on-desk-top-app/

## 说明

本程序可以在桌面上下雪（废话）。在桌面上下雪一直是我想要的一种效果。可惜一直好像都没有我比较喜欢的工具可以达到我心中的想法。所以我就自己开发了一款软件。所谓自己动手，丰衣足食。这个软件做出来后感觉十分欣喜。这个感觉真是太温馨啦（这个感觉应该是动漫里面的那种效果）。

另外还可以双通道调节透明度，改变下落速度（1.1），改变桌面层次和设置开机启动！哈哈，自从上次那朵花动态桌面之后，我就有做这个的想法了，这个软件就继承了那朵花动态桌面的优点啦。推荐使用！！
